type HExcept es = Except (HException es) Source

The heterogeneous exception monad. Computations are either one of a known set of exceptions es or a value.

type HExcept1 e = HExcept (Only e) Source

A specialization of HExcept for computations which can return only one type of exception.

type HExceptT es = ExceptT (HException es) Source

A monad transformer that adds heterogeneous exceptions into other monads.

A specialization of HExceptT for computations which can return only one type of exception.

extend :: (Functor m, Subset es es', TypeIndexed es') => HExceptT es m a -> HExceptT es' m a Source

Extend the given result so that it may be used in a context which can return a superset of the errors that may arise from the original result. This is useful for calling multiple functions which return different errors types from within a single function. Note: this can also be used to re-order the error types.
