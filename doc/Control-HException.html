data HException es Source #

A heterogeneously typed Exception, which holds an underlying exception that is statically constrained to be one of the type-level list es.

The Exception instance for HException es is able to catch any Exceptions thrown whose type is in es. The instance throws the underlying Exception contained in the HException.
